## ðŸ› ï¸ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/FlameyFox/animetools.git
   cd animetools
   ```

2. **Install dependencies:**
   ```bash
   # With npm
   npm install
   # Or with yarn
   yarn install
   # Or with pnpm
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   # With npm
   npm run dev
   # Or with yarn
   yarn dev
   # Or with pnpm
   pnpm run dev
   ```
   The app will be running at [http://localhost:3000](http://localhost:3000)
